NAME

v.in.hecras.py - Read a sdf formatted ascii text file, output from a HEC-RAS analysis and create a polygon vecotr of water surface extent

KEYWORDS

HEC-RAS, water surface

SYNOPSIS

v.in.hecras.py
v.in.hecras.py --help
v.in.hecras.py input=string output=string [--help] [--verbose] [--quiet] [--ui]

Flags:

--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

input=string [required]
Name of input HEC-RAS file (from run of simulation)
output=string [required]
Name of output GRASS polygon vector
