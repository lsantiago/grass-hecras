NAME

v.xsections.py - Creates a river system schematic vector map and a cross sections vector map from a river network

KEYWORDS

vector, HEC RAS, cross sections

SYNOPSIS

v.xsections.py
v.xsections.py --help
v.xsections.py [-s] input=string xsections=string stations=string spacing=integer width=integer intersects=string [smooth_river=string] layer=integer [threshold=integer] [--help] [--verbose] [--quiet] [--ui]

Flags:

-s
Perform smoothing on input map with v.generalize (uses "snakes" algorithm)
--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

input=string [required]
Name of input (river) line vector
xsections=string [required]
Name of output cross sections line vector
stations=string [required]
Name of output river stations schematic point vector
spacing=integer [required]
spacing between cross sections
Default: 100
width=integer [required]
Width of each cross section
Default: 200
intersects=string [required]
Name of output point vector for cross section intersections
smooth_river=string
The smoothed output vector name (only when -s flag is used)
layer=integer [required]
The layer in smoothed output vector
Default: 1
threshold=integer
The threshold for the smoothed vector name (only when -s flag is used)
