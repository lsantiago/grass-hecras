NAME

v.out.hecras.py - Creates a HEC RAS formatted ascii text file of a river system and cross sections from GRASS vector maps

KEYWORDS

vector, elevation profile, HEC RAS, cross sections

SYNOPSIS

v.out.hecras.py
v.out.hecras.py help
v.out.hecras.py river=string xsections=string stations=string elevation=string [resolution=integer] output=string [--verbose] [--quiet]

Parameters:

river=string [required]
Name of input of river schematic line vector (from v.xsections output)
xsections=string [required]
Name of input cross sections line vector (from v.xsections output)
stations=string [required]
Name of input river stations point vector (from v.xsections output)
elevation=string [required]
Name of input elevation raster for making cross section profiles
resolution=integer
Resolution along the cross sections for getting elevation points (if not given, the elevation raster resolution will be used)
output=string [required]
Name of output HEC RAS geometry file (without .sdf extension)
